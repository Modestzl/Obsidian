> [!warning] Важное предупреждение
> API находится в стадии бета-тестирования. В будущем возможны изменения.

> [!info] Требования
> Для работы API необходим токен доступа.


asdfsadf  `sdfasd `



> [!example] Пример запроса
> ```
> curl -X GET https://api.example.com/users
> -H "Authorization: Bearer YOUR_TOKEN"
> ```

> [!success]+ Успешный ответ
> ```json
> {
>   "status": 200,
>   "users": [...]
> }
> ```

> [!danger] Возможные ошибки
> - 401 - Неверный токен
> - 429 - Слишком много запросов
