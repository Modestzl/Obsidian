	##Небольшой пример
```
for ($j = 0; $j < count($goods); $j++) {  
   echo "Имя клиента: {$goods[$j]['name']} </br>";  
    echo "Возраст: {$goods[$j]['age']} </br>";  
    echo "Оценка: {$goods[$j]['score']} </br>";  
  
   echo '<hr>';  
  
 
  
foreach ($goods as $good) {  
    echo "Имя клиента: {$good['name']}: {$good['age']} </br>";  
    echo "Возраст: {$good['score']}  Оценка: {$good['score']} </br>";  
    echo '<hr>';  
}
```
>[!info] массив оператор foreach  не меняет. Надо добавлять тильду &

> [!info]+ Информация
> Это стандартный информационный блок. Текст блока всегда пишется с отступом после ">".

> [!warning] Внимание
> Текст вашего предупреждения.
