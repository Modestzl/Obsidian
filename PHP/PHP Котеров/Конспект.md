
Оператор ??
Как можно видеть из предыдущего раздела, условная инициализация переменной при 
помощи тернарного оператора не является его специализацией. Для решения этой задачи больше подходит отдельный оператор ??. Он позволяет назначить переменной значение только в том случае, если переменная не была ранее проинициализирована или ей присвоено значение null (листинг 9.18).
